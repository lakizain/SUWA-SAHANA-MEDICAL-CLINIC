# SUWA SAHANA MADICAL CLINIC - Clinic Management System

A comprehensive HTML + Bootstrap 5 UI for a clinic management system, featuring a clean, modern, and mobile-responsive design.

## 🏥 Overview

This project provides a complete frontend interface for managing laboratory operations including:

- Patient billing and test management
- Package management for test bundles
- Report generation and analytics
- User administration
- Reference and center management
- Test data management

## 📁 Project Structure

```
lab-new/
├── index.html              # Login page
├── dashboard.html          # Main dashboard with 9-tile navigation
├── package-management.html # Package creation and management
├── reports-management.html # Reports with multiple tabs
├── billing.html           # Comprehensive billing interface
├── admin-users.html       # User management
└── README.md             # Project documentation
```

## 🎨 Design Features

### Bootstrap 5 Components Used

- **Grid System**: Responsive 12-column layout
- **Cards & Panels**: Clean content containers with shadows
- **Tables**: Data display with dark headers
- **Forms**: Floating labels and validation states
- **Buttons**: Consistent styling with icons
- **Navigation**: Tabbed interfaces
- **Modals**: For confirmations and forms
- **Alerts**: Status notifications

### Color Scheme

- **Primary Blue**: #11327c (Headers, buttons, links)
- **Primary Dark**: #162c60 (Hover states, accents)
- **Secondary Teal**: #6ac2a0 (Highlights, badges)
- **Secondary Dark**: #53ab9b (Success actions)
- **Dark**: #242424 (Text, dark elements)
- **Light**: #e6e8e8 (Background)
- **Muted**: #626a7f (Secondary text)
- **Border**: #adc2cf (Borders, dividers)

### Typography

- **Font Family**: Bootstrap default (system fonts)
- **Headers**: Bold weights with proper hierarchy
- **Body Text**: Clean, readable sans-serif
- **Monospace**: For IDs and technical data

## 📱 Responsive Design

The interface is fully responsive and optimized for:

- **Desktop**: Full-featured layout with sidebars
- **Tablet**: Adjusted column layouts
- **Mobile**: Stacked layouts with touch-friendly buttons
